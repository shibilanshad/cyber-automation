# Website Monitoring

This is a simple python script to monitor your websites. 

## Installation and Usage

``` bash
git clone https://github.com/shibilanshad/cyber-automation.git

cd cyber-automation/website-monitoring

pip install -r requirements.txt

```


- Change <YOUR_TELEGRAM_BOT_TOKEN> to your telegram bot api token.
- Change <TELEGRAM_CHAT_ID> to chat id of account you want your messages.
